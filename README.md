# Tristan's Daily

## 本次改动
1. 公式兼容增强：支持 `$$...$$`、`\[...\]`、`$...$`、`\(...\)`。
2. 新增删除功能：文档列表与精选列表可删除，并弹窗提醒“是否删除该文档？”。
3. 切换离开撰写页时，如有未保存内容会提示确认“是否放弃并离开撰写页？”。
4. 增强 Markdown 扩展语法支持：
   - 引用：`> 引用内容`
   - 代码块（```）并支持高亮
   - GFM 表格 / 任务列表 / 删除线
   - 高亮：`==高亮==`
   - 下划线：`++下划线++`

## 存储
- 本地存储：`localStorage`
- 键：`tristan_entries_v4`、`tristan_uploads_v4`
